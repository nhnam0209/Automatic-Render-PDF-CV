# Automatic-Render-PDF-CV
Using Python to render CV PDF 


This project fixes HTML CV problems  overflow text and make your CV looks more beautiful!!!

TO USE THIS:

1. Modify your information in .json file
2. Compile main.py -> they will render a lot of files in folder that you use in main.py
3. Upload all .tex files into overleaf.com 
4. Compile in the website and download as PDF
